# PRODUCT_UI_SPEC.md --- Analog Intelligence™ UI / UX

This document defines the user interface and user experience.

Mockup reference: Analog Intelligence™ -- Negative Scanner interface
design.

Main navigation tabs: - Scan - Gallery - Insights (PRO)

------------------------------------------------------------------------

## Scan Tab

Purpose: Capture film negatives.

UI: - Camera preview - Frame alignment overlay - Capture button -
Settings icon

Free version: - Watermark - Sponsored ad banner

Workflow: 1. Place film negative on light source 2. Align frame 3. Tap
capture 4. App transitions to Adjust screen

------------------------------------------------------------------------

## Adjust Screen

Controls: - Exposure slider - Warmth slider - Contrast slider

Pro options: - AI Color Reconstruction - AI Dust Removal

User presses Done to save.

------------------------------------------------------------------------

## Gallery Tab

Displays scanned images in grid layout.

Features: - Tap to preview - Multi-select - Export - Delete

Pro feature: - Contact sheet generator

------------------------------------------------------------------------

## Insights Tab (Pro)

Provides roll exposure analysis.

Includes: - Histogram chart - Shadow clipping % - Highlight clipping % -
Template-based insights

Example: - Highlights preserved well - Overall well-exposed roll

------------------------------------------------------------------------

## Monetization

Free: - Watermark - Ads - Limited resolution

Pro (\$9.99): - No ads - No watermark - Full resolution export - AI
processing - Insights tab
